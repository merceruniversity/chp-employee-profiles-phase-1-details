A div wraps each of the elements above with a class name based on the Element Name + Wrapper.

e.g. The CHPEmloyeeProfilesPhase1Details-headshot element is wrapped in a div with the class CHPEmloyeeProfilesPhase1Details-headshotWrapper.

JavaScript scans each of the element wrapper divs and adds a class based on has + Element Name to the closest CHPEmloyeeProfilesPhase1Details element if content is found inside of the wrapper.

e.g. If content is found within the CHPEmloyeeProfilesPhase1Details-faxNumberWrapper wrapper div, the class CHPEmloyeeProfilesPhase1Details--hasFaxNumber is appended to the closest CHPEmloyeeProfilesPhase1Details div.

Headshot Dimensions

The headshot is a maximum of 25% of the details view width.

The details view will be in a container that is a maximum of 1200px wide. (Ignoring gutters.)

The page will probably be in two-column mode, but assumptions should not be made.

There are no special, more mobile-friendly, designs used at smaller screen sizes.

I decided to use a 1:1.2 aspect ratio for the headshots

  • 1200px / 4 = 300px
  • 300px * 1.2 = 360px

The maximum headshot dimensions are 300px wide by 360px high

Or, if the device has a 2x pixel density, 600px wide by 720 high
